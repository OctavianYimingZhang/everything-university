# Feedback Sources

Use this reference to collect tutor, TA, marker, and supervisor feedback.

## Source Priority

1. LMS submission feedback and gradebook comments.
2. Rubrics and marking criteria attached to submissions.
3. Inline comments in returned DOCX/PDF files.
4. Turnitin-style feedback reports when the student can access them.
5. Office-hour notes supplied by the user.
6. Email or LMS messages containing feedback.
7. Peer review comments when relevant to the task.

## Normalization

Normalize raw feedback into:

- evidence use;
- argument structure;
- citation style;
- academic tone;
- calculation/method errors;
- data presentation;
- figure/table expectations;
- missed requirements;
- recurring weaknesses;
- explicit marker preferences.

Preserve the raw feedback source separately from inferred takeaways. Mark inferred takeaways as `confidence: inferred`.

## Contradiction Handling

If feedback sources conflict, keep both records and tag the task context. For example, a lab marker preference should not automatically override an essay tutor preference.
